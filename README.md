# Cloud To-Do App

A simple **cloud-based to-do application** using **Firebase Firestore** and **Python**.

---

## Features

- User-specific task management
- Create, Read, Update, Delete (CRUD) tasks
- Stores tasks securely in Firebase Firestore
- Timestamps for task creation and updates
- CLI-based interface for simplicity
